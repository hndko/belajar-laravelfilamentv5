## 3ï¸âƒ£ GitHub Issues (Untuk Bug) ğŸ›

Kamu menemukan **bug** (kesalahan program)? Laporkan di GitHub Issues!

### Cara Melaporkan Bug:

**Paling Mudah:**
Jalankan perintah ini di proyek Laravel-mu:

```bash
php artisan make:filament-issue
```

Ini akan membuat issue baru dengan semua informasi versi yang dibutuhkan!

**Manual:**

1. Buka [GitHub Issues](https://github.com/filamentphp/filament/issues/new/choose)
2. Pilih template yang sesuai
3. Isi informasi yang diminta

### âš ï¸ Penting: Repository Reproduksi

Saat melaporkan bug, kamu perlu membuat **repository reproduksi** - yaitu proyek Laravel sederhana yang menunjukkan bug tersebut.

**Kenapa ini penting?**

- Memudahkan developer untuk melihat masalahnya
- Mempercepat proses perbaikan
- Issue tanpa reproduksi mungkin ditutup!

**Template untuk mulai:**
[https://unitedbycode.com/filament-issue](https://unitedbycode.com/filament-issue)
